# RESPECC — TRAP UNIVERSE

A trippy, psychedelic single-page web experience.  
Animated starfield warp tunnel · Matrix rain · Glitch text · Beat visualizer · Neon hack aesthetic.

## Run

Just open `index.html` in any modern browser — no build step required.

## Features

- **Starfield warp tunnel** — 3D star parallax that explodes in hyperspace mode
- **Matrix rain** — scrolling RESPECC/trap/katakana characters
- **Glitch title** — chromatic aberration glitch animation
- **Beat visualizer** — 32-bar animated EQ, click *DROP BEAT* to randomize
- **Terminal ticker** — scrolling hacker-style status messages
- **Interactive buttons** — DROP BEAT · GLITCH · HYPERSPACE
- **Click ripples** — neon ripple effect on every click
- **Scanlines + vignette** — CRT-style overlay
